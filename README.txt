# LFA-Trabalho1

Trabalho 1 de Linguagens Formais e Autômatos  
Formal Languages and Automata - Assignment 1

## Requisitos / Requisites

`python >= v3.8.5`, `pyyaml >= v5.3.1`

## Uso / Usage

Em um terminal, navegue para o diretório contendo o programa a ser executado:
`FSM/` (Máquina de Estados Finitos) ou `Turing/` (Máquina de Turing).

Ajustes nos dados de entrada e outros parâmetros podem ser feitos editando o
respectivo arquivo YAML no diretório do programa: `FSM_input.yaml` ou
`Turing_input.yaml`.

Execute o programa com o seguinte comando: `python3 FSM.py` ou `python3
Turing.py` e siga as instruções na tela.

--------------------------------------------------------------------------------

In a terminal, navigate to the directory containing the program to be run:
`FSM/` (Finite State Machine) or `Turing/` (Turing Machine).

Adjustments to the input data and other parameters can be made by editing the
respective YAML file in the program's directory: `FSM_input.yaml` or
`Turing_input.yaml`.

Run the program with the following command: `python3 FSM.py` or `python3
Turing.py` and follow the on-screen instructions.
